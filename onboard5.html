üìå Why Use Cloud Functions?

Before you set anything up, understand why Cloud Functions matter:

Firebase Cloud Functions let you run backend (server-side) code without managing servers.

They're perfect for tasks you can't trust the frontend with, like:

  • Keeping your API keys secret (e.g. OpenAI, Stripe, SendGrid)
  • Running secure business logic (e.g. checking payments before updating Firestore)
  • Generating dynamic content ‚Äî like Firestore Security Rules ‚Äî safely
  • Handling heavy tasks (e.g. image processing, sending emails, cron jobs)
  • Creating custom APIs for your web/app

How does it work?

  • You write a function in VS Code
  • Deploy it with Firebase
  • Google Cloud runs it only when needed ‚Äî automatically scaling for you

Result: No servers to manage. No accidental exposure of secrets in your frontend. ‚úÖ

üîπ Part 1: Understand the Blaze Plan

Why You Need Blaze

  • The Spark (Free) plan does not let you deploy real Cloud Functions ‚Äî only run them locally for testing
  • To publish functions to the real backend, you must be on the Blaze (Pay-as-you-go) plan
  • Good news: Google gives you $300 in free credits for new Cloud accounts ‚Äî enough for most learning and small production projects for months

How Billing Works

Plan Base Cost When You Pay Key Limits
Spark (Free) $0/month Never charged No live Cloud Functions
Blaze (Pay-as-you-go) $0/month You only pay if you go beyond free limits Full access to Cloud Functions, Hosting with custom domains, advanced Auth, etc.

üëâ Cloud Functions include a free usage tier:

  • 2 million invocations/month free
  • Up to 40,000 GB-seconds of compute time free
  • 1 GiB outbound networking free

So if you're just testing, you're unlikely to be charged.

How to Add & Pause Autopay

Why pause autopay? So you don't get surprised with charges if you forget to delete a project or your functions go viral overnight.

Sample Steps to Add & Control Autopay:

  1. 1 After adding your card ‚Üí Google Cloud creates a Billing Account for your Firebase project.
  2. 2 Visit the Google Cloud Billing Console.
  3. 3 Select your billing account ‚Üí Go to Payment Settings.
  4. 4 Here you can:
    • Check that your card is added
    • See if autopay is on
    • Click Manage Autopay ‚Üí Set your payment method to manual or pause your billing account entirely if you're done testing

Sample Pro Tip: Add a budget alert:

  • Go to Billing ‚Üí Budgets & Alerts ‚Üí Create a budget
  • Example: Limit = $5 ‚Üí Get an email if your usage crosses that amount

‚úÖ This keeps you 100% safe ‚Äî you'll know before a real charge hits your card.
