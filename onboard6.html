1Install Node.js

üí°Node.js lets you run JavaScript outside the browser. Firebase CLI depends on it.

  1. Go to the official download page: https://nodejs.org/en/download
  2. Click the big green LTS button ‚Äî "Recommended For Most Users"
  3. Download the installer for your OS (Windows / macOS / Linux)
  4. Run the installer:

    • Click Install
    • Click Finish when done

2Create a Project Folder

üìÅThis is the folder where you'll keep all your Firebase code.

  2. Pick a location like Documents or Desktop

3Open Project in VS Code

  1. Open Visual Studio Code
  2. Click File > Open Folder...

4Open Terminal in VS Code

  1. In VS Code, open Terminal:
    • Menu: Terminal > New Terminal
    • Or use shortcut: Ctrl + ` (backtick) on Windows, Cmd + ` on Mac

5Check Node.js Installed Correctly

6Install Firebase CLI

The Firebase CLI lets you talk to Firebase from your computer.

In the terminal, run:

üìù-g means global install ‚Üí usable anywhere.

When done, check it worked:

7Login to Firebase

In the terminal, run:

  2. Click Allow to let Firebase CLI access your Firebase projects

‚úÖ Back in the terminal, you'll see:
